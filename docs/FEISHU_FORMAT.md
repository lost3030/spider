# 飞书通知格式优化说明

## ❌ 旧格式（原始 JSON）

```
```json
{
  "summary": "马斯克转发支持特朗普账户被封，暗示平台政策可能影响其商业决策，短期利空",
  "signal_type": "D",
  "direction": "Short",
  "assets": {
    "US": ["TWTR"],
    "CN": []
  },
  "confidence": 5,
  "expiry": "1天",
  "risk": "若马斯克最终未采取实际行动，信号失效"
}
```
```

**问题：**
- 😖 JSON 格式不易阅读
- 😖 信号类型（A/B/C/D/E）看不懂
- 😖 没有 emoji 图标，信息层级不清晰
- 😖 置信度是数字，需要自己判断高低
- 😖 没有截图链接（或不明显）

---

## ✅ 新格式（友好的富文本）

```
🔔 马斯克推文分析

📝 摘要
马斯克转发支持特朗普账户被封，暗示平台政策可能影响其商业决策，短期利空

🏷️ 信号类型
💬 情绪/口水战

📊 影响方向
📉 做空

💼 受影响资产
🇺🇸 美股：TWTR
🇨🇳 A股：无直接影响

⭐⭐ 中等置信度 (5/10)

⏰ 信号时效：1天

⚠️ 关键风险
若马斯克最终未采取实际行动，信号失效

🖼️ 截图：https://shenyuan-x.oss-cn-hangzhou.aliyuncs.com/2010492174714999291.jpg
```

**优点：**
- ✅ 使用 emoji 图标，层级清晰
- ✅ 信号类型有中文说明（情绪/口水战）
- ✅ 方向用图标表示（📈 做多 / 📉 做空）
- ✅ 置信度分级（⭐⭐⭐ 高 / ⭐⭐ 中 / ⭐ 低）
- ✅ 截图链接明显标注
- ✅ 移动端友好，易读易理解

---

## 🎨 格式化规则

### 信号类型映射
- `A` → 📊 行动/公司行为
- `B` → 🏛️ 政策立场
- `C` → 🚀 技术突破/产品发布
- `D` → 💬 情绪/口水战
- `E` → 🎮 纯娱乐

### 影响方向
- `Long` → 📈 做多
- `Short` → 📉 做空
- `Neutral` → ➖ 中性

### 置信度评级
- `7-10` → ⭐⭐⭐ 高置信度 (可操作)
- `4-6` → ⭐⭐ 中等置信度 (需观察)
- `0-3` → ⭐ 低置信度 (噪音)

### 资产标注
- 🇺🇸 美股
- 🇨🇳 A股
